## Journal-App 

The project was created using Vite.  

Stack used to develope this SPA app:  

 
#
**Frontend:**  
* React18 + Hooks  
* Redux + Redux-Toolkik  
* Material UI - HTML - CSS
#
**Backend & Database:**  
* Firebase - Firestore Database
#
**Images container:**  
* Cloudinary
#
**Testing Tools:**  
* Jest, React Testing Library
